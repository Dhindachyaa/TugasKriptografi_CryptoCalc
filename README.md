
# 🔐 CryptoCalc
A modern, sleek web-based cryptography tool built with **Vue 3**.  
Encrypt and decrypt text using classic and historical cipher algorithms with a futuristic glassmorphism UI.

---

## 🚀 Tech Stack
- **Framework**: Vue.js 3 (Composition API)
- **Styling**: CSS3 (Glassmorphism + Keyframe Animations)
- **Fonts**: Space Mono & DM Sans

---

## 🛠️ Installation & How to Run
Follow these steps to run the project locally:
### 1️⃣ Clone Repository
```bash
git clone https://github.com/yourusername/crypto-calc.git
cd crypto-calc
````
### 2️⃣ Install Dependencies
Make sure **Node.js** is installed on your computer.
```bash
npm install
```

### 3️⃣ Run Development Server
Start the development server:
```bash
npm run dev
```
The app will usually run at:
```
http://localhost:5173
```
